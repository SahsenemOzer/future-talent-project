# future-talent-project

Simple QR-powered portfolio builder for Future Talent application.

## What it does

- You enter profile, education, skills, and projects.
- App generates a short share URL and QR code.
- When scanned, the target device opens directly in portfolio view.

## Run

Open `index.html` locally for editing/testing.

For QR scanning on mobile, publish with GitHub Pages or Vercel and use the live URL.
